# Daisy

### Branch convention

| 브랜치  |                    비고                    |
| :-----: | :----------------------------------------: |
|  main   |             배포 완료시 브랜치             |
| develop |                개발 브랜치                 |
| feature |    기능 브랜치, 규칙은 feature/기능이름    |
| release | 배포시 브랜치, 배포 완료후에는 main에 병합 |

### Commit convention

|   커밋   |       비고        |
| :------: | :---------------: |
|  build   |      빌드 시      |
|  chore   |   패키지설치 등   |
|   docs   | 문서 추가 및 변경 |
|   feat   |     기능 추가     |
|   fix    |     버그 수정     |
| refactor |     리팩토링      |
|  style   |    스타일 변경    |
|   test   |      테스트       |
